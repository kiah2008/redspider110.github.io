Java Lambda 表达式

发表于 2018-04-04 | 分类于 Java

概念

  • Lambda
    大写 Λ，小写 λ。读音：lan b（m） da（兰木达）['læmdə]。
  • Lambda 表达式
    Lambda expression ：基于数学中的 λ 演算得名，直接对应于其中的 Lambda 抽象(lambda abstraction)，是一个匿名函数，即没有函数名的函数，在 Java 中又称为闭包或匿名函数。而 λ演算 是函数式编程的基础，所以 Lambda 表达式具有部分函数式语言的特征。特点就是简(省)单(代)优(码)雅。
